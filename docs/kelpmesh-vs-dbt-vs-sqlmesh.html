KelpMesh Core & Studio vs dbt vs SQLMesh — 90-feature matrix

KelpMesh is split by install layer. Core is fully free and Apache 2.0. Studio is freemium.

Two packages. One engine.
pip install kelpmesh-core — full CLI engine, all warehouses, 32 macros, scheduler, security. Always free. Apache 2.0.
pip install kelpmesh-studio — everything in Core plus the FastAPI browser dashboard. Free for personal projects (1 user, 3 projects). Pro $29/user/mo for commercial teams (RBAC, API keys, git sync, AI). Business $79/user/mo adds SSO and BYOC.

Studio columns marked ⊇ Core inherit every Core engine feature — the Studio column only highlights what the browser layer adds.
